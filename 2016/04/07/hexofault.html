hexo+github搭建独立博客遇到的问题及处理几则

By Whatbeg

发表于 2016-04-07

文章目录

  2. 2. 错误1: 安装hexo时出现 ‘operation not permitted’
  3. 3. 错误2: hexo init时出现 ‘ERROR Local hexo not found in ..’

用hexo+github搭建个人博客时出现过一些错误，有一些已经忘记了，把记得的两个记录一下，以防以后系统重装后又要搭建时要到处找解决方法。

搭建工具

  • Windows 8.1 -64bit
  • nodejs 5.9.0 -64bit
  • Git 2.8.0 -64bit

错误1: 安装hexo时出现 ‘operation not permitted’


解决方法：
没有权限，可以在开始菜单找到cmd（命令行），右键以管理员身份运行，这时候就有root权限了，然后在cmd中转到你的hexo目录下(如D:\hexo），再运行npm install -g hexo即可

错误2: hexo init时出现 ‘ERROR Local hexo not found in ..’


解决办法：
到这一步，其实说明hexo没有安装成功，办法就是再像上个错误那样，进入管理员cmd，再安装一下hexo即可，建议npm install也在cmd中做了。
出现的那些WARN 其实不用管它，是有一个包没有安装成功，因为不适合此操作系统，但其实不影响。

暂时只记得这么多了，以后再遇到问题再加上来。
